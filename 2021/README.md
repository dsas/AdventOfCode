# Advent of code 2020
Solutions for [Advent of Code 2021](https://adventofcode.com/2021) challenge in javascript.


# Install
```
git clone https://github.com/dsas/AdventOfCode.git
cd AdventOfCode/2021
yarn install
```
# Test
```
yarn run test
yarn run lint
```

# Run
```
yarn run

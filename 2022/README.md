# Advent of code 2022
Solutions for [Advent of Code 2022](https://adventofcode.com/2022) challenge in javascript.


# Install
```
git clone https://github.com/dsas/AdventOfCode.git
cd AdventOfCode/2022
yarn install
```
# Test
```
yarn run test
yarn run lint
```

# Run
```
node . <Day no>
```
